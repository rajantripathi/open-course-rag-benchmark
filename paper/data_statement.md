# Data Statement

## Sources
All corpus materials are drawn from openly licensed OpenStax textbooks:
Principles of Data Science (CC BY-NC-SA 4.0) and Introduction to Philosophy
(CC BY 4.0). No proprietary, paywalled, or institutionally restricted
materials are used.

## LLM training exclusion
Both source textbooks include a clause prohibiting use in training large
language models. This study uses the texts exclusively as a retrieval corpus
for evaluation purposes. No model weights are updated using this text. The
texts serve as static documents against which retrieval systems are tested.

## No human subjects
This benchmark uses synthetically generated and manually curated questions.
No student data, interaction logs, or personally identifiable information
is collected or used. No ethics approval is required.
